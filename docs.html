API Endpoints

GET /api/tags

Fetch all available quote tags from Goodreads (cached for 1 hour).

Response

{
  "tags": ["love", "life", "inspiration", "wisdom", ...]
}
GET /api/qotd

Get the Quote of the Day.

Response

{
  "quote": "Be yourself; everyone else is already taken.",
  "author": "Oscar Wilde"
}
GET /api/quotes?page=1&tag=love

Fetch paginated quotes with optional tag filter.

Parameters

Parameter Type Description
page integer Page number (default: 1)
tag string Filter by tag (optional)

Response

{
  "quotes": [
    {
      "text": "Quote text here",
      "author": "Author Name"
    }
  ]
}
GET /

Serves the frontend HTML application.

Usage Examples

Fetch All Tags

// JavaScript Fetch API
fetch('http://localhost:5000/api/tags')
  .then(res => res.json())
  .then(data => console.log(data.tags));

Get Quote of the Day

// Using async/await
async function getQOTD() {
  const response = await fetch('/api/qotd');
  const data = await response.json();
  console.log(`${data.quote} - ${data.author}`);
}

getQOTD();

Paginated Quotes with Tag Filter

// Fetch love quotes from page 3
const page = 3;
const tag = 'love';

fetch(`/api/quotes?page=${page}&tag=${tag}`)
  .then(res => res.json())
  .then(data => {
    data.quotes.forEach(quote => {
      console.log(`"${quote.text}" - ${quote.author}`);
    });
  });

Auto-refresh Implementation

// Refresh quotes every 60 seconds
function autoRefresh() {
  setInterval(() => {
    fetchQuotes(currentPage, selectedTag);
  }, 60000); // 60 seconds
}

autoRefresh();

Typography System

Quote of the Day

  • Font: Georgia serif, italic
  • Size: Fixed 1.15rem (consistent regardless of length)
  • Line-height: 1.8
  • Letter-spacing: 0.3px
  • Alignment: Center
  • Purpose: Professional, consistent appearance for featured quote

Regular Quotes

  • Font: Georgia serif, italic
  • Size: Responsive clamp(14px, 2.8vw, 16.5px)
  • Line-height: Responsive clamp(1.45, 1.55, 1.65)
  • Letter-spacing: 0.15px
  • Behavior: Adapts fluidly to viewport width

Mobile Optimization

  • Base font size: 16px (prevents iOS auto-zoom)
  • Text size adjust: 100% (both webkit and standard)
  • Word-break: break-word (prevents overflow)
  • Text-align: center (improved readability)

Responsive Typography Formula

/* HTML base */
html { font-size: 16px; }

/* Body */
body {
  font-size: 1rem;
  line-height: 1.6;

/* Fluid scaling with clamp() */

  font-size: clamp(14px, 2.8vw, 16.5px);
  line-height: clamp(1.45, 1.55, 1.65);
}

Theme Toggle Button

Switches between light and dark mode with localStorage persistence.

function toggleTheme() {
  const current = document.documentElement.getAttribute('data-theme');
  const next = current === 'dark' ? 'light' : 'dark';
  document.documentElement.setAttribute('data-theme', next);
  localStorage.setItem('theme', next);
}

Architecture

Tech Stack

Layer Technology Purpose
Backend Node.js + Express Web server and API
Scraping Axios + Cheerio Fetch and parse Goodreads HTML
Frontend Vanilla HTML/CSS/JS Single-page application
Styling CSS Variables + clamp() Theming and responsive design
Data Source Goodreads.com Quote content

Data Flow

User Request ‚Üí Express Server ‚Üí Web Scraper (Cheerio)
     ‚Üì
Goodreads HTML ‚Üí Parse Quotes/Tags ‚Üí Cache (1 hour)
     ‚Üì
JSON Response ‚Üí Frontend ‚Üí Render UI ‚Üí User Display

Caching Strategy

  • Tags: Cached for 1 hour to reduce server load
  • Quotes: Fresh fetch on each request for variety
  • QOTD: Rotates daily based on date
  • Theme: Stored in browser localStorage

Performance Optimizations

Frontend

  • CSS clamp() for fluid typography (no JS calculations)
  • CSS transitions instead of JS animations
  • Debounced scroll events
  • LocalStorage for theme preference

Backend

  • Tag endpoint caching (1 hour TTL)
  • Efficient Cheerio selectors
  • Async/await for non-blocking I/O
